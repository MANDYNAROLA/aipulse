# AIPulse — Live AI Intelligence Network

> Real-time monitoring dashboard for 9 AI platforms. Built with Firebase + GitHub Pages.

🌐 **Live site:** https://MANDYNAROLA.github.io/aipulse

---

## What it does

- **9 AI platforms tracked live** — ChatGPT, Gemini, Claude, Grok, Perplexity, Copilot, DeepSeek, Meta AI, Mistral
- **Real-time latency** with p50/p95/p99 percentiles
- **Community reports** — powered by Firebase Realtime Database
- **Quality scores** — community rated per AI platform
- **Smart work window** — best time to use AI today
- **GitHub auto-commit** — connect once, get daily commits automatically
- **Social layer** — follow users, react to reports
- **Add any AI** — suggest new platforms to track

## Tech stack

- Pure HTML/CSS/JS — no framework needed
- Firebase Authentication + Realtime Database
- GitHub Pages hosting (free)
- Chart.js for latency visualization

## Run locally

Just open `index.html` in any browser. No build step needed.

## Firebase setup

See the config already embedded in `index.html`. Replace with your own Firebase project config if forking.
